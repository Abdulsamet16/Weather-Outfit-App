# 🌦️ Weather & Outfit Suggestion App

🔗 Live Demo: https://outfit-weather-app.netlify.app

A responsive weather application that fetches real-time weather data and suggests outfit recommendations based on temperature and weather conditions. Built using vanilla JavaScript and OpenWeather API.

---

## 🚀 Features

-  Search weather by city
-  Get weather using your current location (Geolocation API)
-  Outfit suggestions based on temperature
-  Weather-based advice (rain, snow, clear, etc.)
-  Dark mode toggle
-  Fully responsive design (mobile-friendly)

---

## Built With

- HTML5
- CSS3 
- JavaScript
- OpenWeather API
